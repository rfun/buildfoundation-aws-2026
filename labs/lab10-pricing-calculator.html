Cost Optimization

Lab 10: AWS Pricing Calculator and Trusted Advisor

⏱ ~20 minutes ✅ Free Tier 📚 AWS Console Only

Overview

Understanding costs before you build is a key Cost Optimization practice. The AWS Pricing Calculator lets you estimate costs for architectures before deploying them, while AWS Trusted Advisor provides automated recommendations to optimize your existing environment. In this lab, you'll estimate costs for a sample 3-service architecture and explore Trusted Advisor's free checks. This connects to the Cost Optimization pillar principles "Adopt a consumption model", "Measure overall efficiency", and "Stop spending money on undifferentiated heavy lifting" from the Week 2 lecture.

Objectives

  • Use the AWS Pricing Calculator to estimate costs for a multi-service architecture
  • Understand the cost impact of instance right-sizing (t3.micro vs t3.medium)
  • Explore AWS Trusted Advisor's free checks across all pillar categories
  • Learn how automated recommendations help optimize cost, security, and performance

Prerequisites

  • An AWS account (free tier is sufficient)
  • Signed in to the AWS Management Console
  • A web browser (Pricing Calculator is a separate web tool)

Step-by-Step Instructions

1
Open the AWS Pricing Calculator.
In a new browser tab, go to calculator.aws (or search "AWS Pricing Calculator" in your browser). Click "Create estimate" to start building your cost estimate.
2
Add an Amazon EC2 instance.
Click "Add service" → search for Amazon EC2 → select it. Configure the following:
  • Region: US East (N. Virginia)
  • OS: Linux
  • Instance type: t3.micro
  • Quantity: 1
  • Usage: 730 hours/month (this means running 24/7)
  • Leave other settings as defaults → click "Save and add service"

Note the monthly cost estimate for EC2.

3
Add Amazon S3 storage.
Click "Add service" → search for Amazon S3 → select it. Configure the following:
  • Region: US East (N. Virginia)
  • S3 Standard storage: 50 GB per month
  • GET requests: 10,000 per month
  • PUT requests: 1,000 per month
  • Leave other settings as defaults → Save and view summary
4
Add Amazon DynamoDB.
Click "Add service" → search for Amazon DynamoDB → select it. Configure the following:
  • Region: US East (N. Virginia)
  • Table class: Standard
  • Capacity mode: On-demand
  • Average item size: 1 KB
  • Read request units: 25 per second
  • Write request units: 25 per second
  • Save and view summary
5
Review the total monthly estimate.
Review the total monthly estimate for all three services. This is a simple web application architecture: compute (EC2), storage (S3), and database (DynamoDB).
6
Demonstrate the impact of right-sizing.
Go back to the EC2 configuration → change the instance type from t3.micro to t3.medium → observe how the monthly cost changes. This shows why choosing the right instance size matters for cost optimization!
7
Navigate to Trusted Advisor.
Switch to the AWS Console → use the search bar and type Trusted Advisor. Click on it to open the Trusted Advisor dashboard.
8
Explore the available checks in each category.
  • Cost Optimization: Look for recommendations about idle resources, underutilized instances
  • Security: Check for open security groups, MFA on root account, public S3 buckets
  • Fault Tolerance: Review warnings about single-AZ deployments, backups
  • Performance: Review suggestions about overutilized instances
  • Service Limits: Check your current usage vs account limits
9
Review the Trusted Advisor dashboard summary.
Note the green (OK), yellow (warning), and red (action recommended) indicators for each category. These provide a quick visual overview of your account's optimization status.
Note: Not all Trusted Advisor checks are available on the Basic (free) support plan. You'll see some checks marked as requiring a Business or Enterprise support plan. Focus on the free checks available — they still provide valuable insights.

Cleanup

🗑 Cleanup Steps:

No cleanup needed!

  • The AWS Pricing Calculator is a web-based tool — your estimate is in the browser only
  • Trusted Advisor is read-only — it only shows recommendations, it doesn't change anything

Key Takeaways

  • Always estimate costs before building — the Pricing Calculator prevents surprises in your AWS bill
  • Right-sizing (choosing the correct instance type) can dramatically reduce costs with no performance penalty
  • Trusted Advisor automates the "measure overall efficiency" principle by continuously checking your environment
  • The combination of planning (Pricing Calculator) and monitoring (Trusted Advisor) implements a complete cost optimization lifecycle
