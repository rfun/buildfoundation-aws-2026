⚙ Operational Excellence
Lab 1
Exploring CloudTrail Event History
⏱ ~15 min • CloudTrail
Explore the audit trail of all API activity in your AWS account. Learn to filter events and read event records.
Lab 2
Building a CloudWatch Dashboard with a Billing Alarm
⏱ ~25 min • CloudWatch, SNS
Create a CloudWatch dashboard with billing metrics and set up an alarm to notify you of unexpected charges.
🔒 Security
Lab 3
IAM Users, Groups, and Least Privilege Policies
⏱ ~20 min • IAM
Create IAM users and groups, attach policies, and use the Policy Simulator to test least privilege access.
Lab 4
Securing an S3 Bucket with Encryption and Access Controls
⏱ ~25 min • S3
Create a secure S3 bucket with encryption, versioning, public access blocking, and an HTTPS-only bucket policy.
🛡 Reliability
Lab 5
S3 Versioning and Object Recovery
⏱ ~15 min • S3
Enable S3 versioning to protect against accidental deletions and practice recovering deleted objects.
Lab 6
Exploring VPC Infrastructure and Availability Zones
⏱ ~20 min • VPC
Explore VPC components, subnets across Availability Zones, route tables, and create a security group.
⚡ Performance Efficiency
Lab 7
Exploring EC2 Instance Types and Launching an Instance
⏱ ~25 min • EC2
Compare EC2 instance families, launch a free-tier instance, and explore monitoring metrics.
Lab 8
Creating and Querying a DynamoDB Table
⏱ ~20 min • DynamoDB
Create a DynamoDB table, add items, run queries and scans, and explore capacity modes.
💰 Cost Optimization
Lab 9
Setting Up AWS Budgets and Free Tier Tracking
⏱ ~20 min • AWS Budgets, Billing
Create budgets to proactively monitor spending, explore the Billing Dashboard, and track free tier usage.
Lab 10
AWS Pricing Calculator and Trusted Advisor
⏱ ~20 min • Pricing Calculator, Trusted Advisor
Estimate costs for a sample architecture and explore Trusted Advisor recommendations for optimization.
🌱 Sustainability
Lab 11
Configuring S3 Lifecycle Policies for Efficient Data Management
⏱ ~15 min • S3
Set up lifecycle policies to automatically transition data to cheaper storage and reduce environmental impact.
Lab 12
Running an AWS Well-Architected Framework Review
⏱ ~25 min • Well-Architected Tool
Use the Well-Architected Tool to conduct a framework review, focusing on sustainability questions and improvement plans.
